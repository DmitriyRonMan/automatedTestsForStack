# Автоматизированные тесты для проверки добавления записи в таблицу, используя открытый интернет-ресурс Стэк IT

## Документация

- [Отчёт о проведённом тестировании](https://github.com/DmitriyRonMan/automatedTestsForStack/blob/main/docs/Report.md)

## Начало работы

### ПО для работы с проектом

Перед началом работы убедитесь, что на вашем устройстве установлены следующие ПО:
1. IntelliJ IDEA или другой инструмент для работы.
2. GIT
3. Браузер - Google Chrome
4. Node.js

### Установка и запуск

Склонировать репозиторий на свой рабочий стол
```
git@github.com:DmitriyRonMan/automatedTestsForStack.git
```

Открыть проект в вашем инструменте

Для запуска тестов использовать команду
```
npx playwright test
```

Для получения отчета использовать команду 
```
npx playwright show-report
```
